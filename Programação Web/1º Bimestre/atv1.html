HTML

O HTML é o componente básico da web, que estabelece a estrutura básica de um website.

HTML é a mais recente evolução do padrão que define a linguagem HTML.

A versão HTML 5 trouxe novas mudanças e funções de acessibilidade que facilitam seu uso.

Trouxe novas tags semânticas, que permite uma estrutura mais clara ao texto.
Outra capacidade do HTML 5 é de criar aplicativos da web offline, onde seus dados são salvos localmente no navegador.

História

Esse componente foi iniciado em 1989, onde Tim Berners-Lee, um cientista da computação, teve a ideia da World Wide Web.

Existiram 4 versões antes do HTML 5, desde 1991 até 1999 e em 2008, a primeira versão pública foi lançada do HTML 5. Essa versão trouxe simplicidade para as construção de websites, com a criação de seções comuns e específicas como rodapé, cabeçalho, slidebar, menus etc.
